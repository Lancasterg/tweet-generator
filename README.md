# Tweet Generator
## Description
Generate tweets in the style of a user, using a simple recurrent generative model.

## Twitter API keys
Create a new file called `keys`, in the format: 

```
consumer_api_key

consumer_secret_key

access_token

access_secret_token
```
## To train

`python train_model.py -u username`
## To generate tweets
`python generate_tweets -n 10`
